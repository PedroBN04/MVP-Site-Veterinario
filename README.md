# VetClinic App

Este é o projeto front-end para o sistema de clínica veterinária, construído com React, Vite e Tailwind CSS.

## 🚀 Como rodar o projeto na sua máquina

Siga os passos abaixo para configurar e rodar a aplicação localmente:

### 1. Pré-requisitos
Você precisa ter o **Node.js** instalado no seu computador.
- Baixe e instale a versão LTS do [Node.js](https://nodejs.org/).

### 2. Passo a passo

Abra o terminal (ou Prompt de Comando/PowerShell) na pasta raiz deste projeto (`vetclinic-app`) e execute os seguintes comandos:

**Passo A: Instalar as dependências**
Isso irá criar a pasta `node_modules` com todas as bibliotecas necessárias.
```bash
npm install
```

**Passo B: Iniciar o servidor de desenvolvimento**
Isso irá compilar o projeto e iniciar um servidor local.
```bash
npm run dev
```

**Passo C: Acessar no navegador**
Após rodar o comando acima, o terminal mostrará um link (geralmente `http://localhost:5173/` ou similar). Clique no link ou copie e cole no seu navegador para ver o sistema funcionando!

### 💻 Tecnologias Utilizadas
- React
- Vite
- Tailwind CSS
- React Router Dom
- Lucide React (Ícones)
